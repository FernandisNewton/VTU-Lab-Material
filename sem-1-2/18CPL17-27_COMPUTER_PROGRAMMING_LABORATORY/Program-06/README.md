# Program 06

Implement using functions to check whether the given number is prime and display appropriate messages. (No built-in math function)

## Output

```shell 
$ gedit program-06.c
$ gcc program-06.c

$ ./a.out
Enter the number: 91
91 is not a prime.

$ ./a.out
Enter the number: 20050
20050 is not a prime.

$ ./a.out
Enter the number: 17
17 is a prime.

```
